# Advanced Ticket Booking System 🚂

A robust Java application for managing train reservations, integrating advanced programming concepts and real-time communication.

## 🌟 Key Features
* **Advanced OOP:** Full implementation of inheritance, interfaces, and complex data models (Passenger, Train, Ticket).
* **Real-time Chat:** A Socket-based networking system with a Server-Client architecture for instant messaging.
* **Modern GUI:** A styled user interface built using **JavaFX** and customized with **CSS**.
* **Persistence:** Integrated with **Oracle Database** for secure data storage.

## 🛠️ Tech Stack
* **Language:** Java.
* **Framework:** JavaFX.
* **Database:** Oracle DB.
* **Networking:** Java Sockets.
